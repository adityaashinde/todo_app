## Installation

To get started with the Todo App, follow these steps:

1. **Clone the repository:**
    ```bash
    git clone https://github.com/adityaashinde/todo_app.git
    ```
2. **Navigate to the project directory:**
    ```bash
    cd todo_app
    ```
3. **Install the dependencies:**
    ```bash
    flutter pub get
    ```
4. **Run the app:**
    ```bash
    flutter run
    ```
